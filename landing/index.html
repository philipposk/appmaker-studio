What you get

More than a code generator

Most tools spit out a zip and wish you luck. AppMaker generates, previews, validates, and repairs.

Streaming generation

Files appear as the model writes them — atomic file operations, not one giant blob at the end. You see progress in real time.

In-browser preview

A real Node runtime runs inside your browser (WebContainer). Your generated app boots and runs live — no local install, no server.

Self-correcting AutoFix

It installs, type-checks, lints, and tests the output — then feeds errors back to the model and patches itself until the build is green.

Any LLM you want

Groq, OpenAI, Anthropic, OpenRouter, Together, Mistral, or a local Ollama model. Swap providers per request with automatic fallback.

Tests included

Generated apps ship with a test suite. The AutoFix loop runs them, so what you download actually passes — not just compiles.

Iterate & deploy

Refine with follow-up prompts, browse every version, edit the code or visual layout, then export a zip or push to deploy.

Run it yourself in two minutes

React + TypeScript frontend, Node + Express backend, MongoDB. Clone it, add a key, go.

# clone the repo
git clone https://github.com/philipposk/AppMaker-vibecode.git
cd AppMaker-vibecode

# install both halves
cd backend && npm install
cd ../frontend && npm install

# add your keys to backend/.env (MONGODB_URI, JWT_SECRET, GROQ_API_KEY…)
# then run both
cd backend && npm run dev      # API on :8000
cd frontend && npm start       # UI  on :3000

✓ open http://localhost:3000 and start building
